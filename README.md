This project is about learning and implementing tree-based machine learning models.
The main objective is to understand how Decision Trees and Random Forest algorithms work.

In this task, I used Python along with libraries like Scikit-learn, Pandas, NumPy, and Matplotlib.
The dataset used in this project is the Breast Cancer dataset, which is available in Scikit-learn.

First, the dataset was loaded and converted into a DataFrame for better understanding.
Then, the data was divided into training and testing sets using an 80:20 ratio.

After that, a Decision Tree Classifier model was trained using the training data.
The model was tested, and accuracy was calculated.

To reduce overfitting, the depth of the tree was limited using the max_depth parameter.
The performance of the limited tree was compared with the normal tree.

Next, the Decision Tree was visualized using a graphical representation.
This helped in understanding how the model makes decisions.

Then, a Random Forest Classifier was trained.
The accuracy of the Random Forest model was compared with the Decision Tree model.

Feature importance was calculated to identify which features are most important for prediction.
The important features were also visualized using a graph.

Cross-validation was performed to check the reliability of the model.
The average accuracy was calculated using multiple folds.

The results showed that the Random Forest model performed better than the Decision Tree.
It also reduced overfitting and gave more accurate predictions.

From this task, I learned that Decision Trees are simple but can overfit easily.
Random Forest improves performance by combining multiple trees.

Overall, this project helped me understand tree-based models and their practical applications.
